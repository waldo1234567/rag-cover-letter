# AI Cover Letter Generator

Generate personalized cover letters using AI based on your CV and job descriptions.

## Features
-   PDF CV parsing with section tagging
-   AI-powered personalized cover letter generation
-   RAG-based retrieval for relevant CV content
-   Chinese translation support
-   Text humanization

## Usage

1. Upload your CV (PDF format)
2. Paste company profile and job description(as well as job title)
3. Click "Generate Cover Letter"
4. Copy the generated letter
